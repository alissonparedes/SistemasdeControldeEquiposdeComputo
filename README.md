# SistemasdeControldeEquiposdeComputo

Aplicación WinForms C# con Entity Framework Core para gestión de inventario de equipos.

## Características
- CRUD completo: Equipos, Departamentos, Mantenimiento
- Reportes por departamento
- Relaciones FK implementadas

## Instalación
1. Clonar repositorio
2. `dotnet restore`
3. Configurar connection string
4. `Add-Migration InitialCreate`
5. `Update-Database`
